BeSTspeak v1.01
----------------------------------------------------------------------------------

This is a very straight forward text to speech speak window program.

It is meant to be easy for people with visual impairments to use and otherwise.


Here are the hot keys:
----------------------------------

ESC      - Exit program
F1       - Help
F2       - Describe program credits
F3       - Decrease volume
F4       - Increase volume
F5       - Speak text
F6       - Stop speaking
F7       - Reset synthesizer
F8       - Decrease speaking rate
F9       - Increase speaking rate
F10      - Decrease pitch
F11      - Increase pitch
Ctrl+F11 - Change voice backwards
Ctrl+F12 - Change voice forwards
